# Approval Module

## Current Capabilities

- diff-based approval queue
- review item selection
- approve / reject / defer
- replay-safe queue persistence

## Current UX

- `/queue`
- `/pick <n>`
- `/review`
- `/approve`
- `/reject`
- `/defer`
